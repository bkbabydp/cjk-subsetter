= subsetter

![Logo](https://raw.github.com/hxgdzyuyi/AlexPic/master/skin/alexpic_m.png)

可以在网页中使用中文字体的 Rack 程序。

=== 原理

通过 sfntly (code.google.com/p/sfntly/) 来切割中文字体

=== 依赖

- Java 环境
- Ruby

=== 感谢以下开源项目

- sfntly https://code.google.com/p/sfntly/

=== 浏览器支持

生成 woff 和 ttf 两种字体
